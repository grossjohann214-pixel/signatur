# Customer Link

Fluxo de validação para clientes via magic link.

## Features

- Confirmação de dados
- Assinatura em tela
- Captura de selfie (opcional)
- Geolocalização (opcional)
- Criação de wallet SGL
- Exibição de protocolo

## Development

```bash
pnpm dev
```

Acesse: http://localhost:3001
